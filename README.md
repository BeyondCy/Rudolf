# Rudolf
The Multiplatform Linux Sandbox (support x86 x86-64 arm mips mipsel)

Write by Python3

## Feature

basic: filename filetype filesize strings elf_info hash(md5 sha256 crc32 ssdeep) packer

static: Yara Virustotal

dynamic: strace_log pcap
